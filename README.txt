# Chess

1) Compile and Run Server.java.
2) Compile and Run ChessGame.java on both client-computers. (¹) 
3) On server-computer open any browser and type on search "my ip address", copy your address.
4) On both client-computers write the server-computer ip address, choose your nickname. (²)

(¹) Make sure both client-computers are connected to the server-computer using LAN or connected to the same router
(²) Ensure they are connected properly using chat and see if shapes are moving correctly

* The game has bugs with connections, and possible images will be not loaded properly
